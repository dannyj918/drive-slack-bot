# Using the @help me Bot

@help me searches our Metalios Shared Drive for you. Whenever you have a question about a document or resource — finding a file, or asking what's inside one — ask it in plain English instead of digging through Drive folders yourself.

## 1. Two ways to ask

**A) Direct message** — search the top bar for "help me" or find it under **Apps** in the sidebar, and open a DM. Type your question there. This is fully private — nobody else sees your question or the reply. The bot isn't added to any channels, so @mentioning it won't work.

**B) `/help` slash command** — type `/help` followed by your question in any channel. Only you see your question and the reply — nobody else in the channel sees either one.

## 2. What to expect

- A 👀 emoji shows up first — that means it received your question and is searching.
- It replies with either:
  - **Direct links** to the matching file(s) in Drive, or
  - **An answer pulled from inside a document** (with a note on which file it came from), if your question is about content rather than just finding a file.
- A ✅ shows up when it's done.

## 3. Tips for better results

- Be specific about what you're looking for — the document type, topic, or what's in it.
- If the first answer isn't quite right, just reply with more detail and it'll search again.
- It only searches the Metalios Shared Drive — not your personal Drive, email, or other tools.
- If it says it couldn't find anything, try simpler or different keywords. Sometimes the file just isn't named what you'd expect.

## 4. Troubleshooting

| Problem | What to do |
|---|---|
| No reply at all | Make sure you're messaging it directly (DM) or using `/help` in a channel — @mentioning it won't work. |
| "Couldn't find anything" | Rephrase with different keywords, or add more detail about the document. |
| Found the wrong file | Reply with more context — the more specific, the better the match. |
| Still stuck | Ping Danny. |
